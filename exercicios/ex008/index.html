Principais formatacoes

Negrito ou destaque:

Nessa frase, temos um termo em destaque usando a tag <strong>.

Italico ou ênfase:

Nessa frase, temos um termo com ênfase ou italico usando a tag<em>.

Texto marcado:

Podemos criar tambem um texto marcado usando a tag <mark>.

E nesse paragrafo temos outro texto marcado.

Textos pequenos:

Para deixar um texto pequeno em uma frase, basta usar a tag <small>.

Texto deletado:

Podemos marcar um texto como deletado (riscado), basta usar a tag <del>

Texto inserido ou sublinhado:

Podemos marcar um texto como inserido (sublinhado), basta usar a tag <ins>.

Texto sobescrito:

Para inserir coisas do tipo: X2 + 3, basta usar a tag <sup> no elemento que voce deseja sobescrever.

Texto subscrito:

Para inserir coisas do tipo: H2O, basta usar a tag <sub> no elemento que voce deseja subscrever.
