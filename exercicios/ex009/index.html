Lista de definicoes:

Para criar uma lista de definicoes, basta usar a tag <dl>

As listas de definicao recebem uma palavra atraves da tag <dt> e depois o significado dessa palavra atraves da tag <dd>.

HTML:
Linguagem de marcacao para a criacao do conteudo de sites.
CSS:
Linguagem de marcacao para a criacao do design de um site.
JavaScript
Linguagem de programacao para a criacao de interatividade de um site.
