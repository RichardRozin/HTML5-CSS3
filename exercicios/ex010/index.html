Trabalhando com links

Para criar um link (ancora) em HTML, basta usar a tag <a>, e colocar um link na parte 'href'.

Usando links externos:

Para criar um link externo, basta usar a tag <a> e dentro de 'href' basta colocar o link do site que deseja abrir.

Porem, por se tratar de um link EXTERNO, devemos colocar o complemento 'target' apos o link, para que o site abra em uma nova guia.

Usando links internos

Para criar um link interno, basta usar a tag <a> e, diferente dos links externos, dentro de 'href' deve-se colocar a pasta ou arquivo
que voce deseja abrir.

EX: Esta é a primeira pagina do site! Se voce quiser visitar a segunda pagina, basta clicar aqui.
