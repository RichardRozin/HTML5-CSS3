Imagens que se adaptam sozinhas

Na hora de criar um site, devemos lembrar que nao sera so um tipo de aparelho que ira acessa-lo.
por isso, usar o conceito de imagens dinamicas deve vir a caso. Utilizar imagens dinamicas em um site Ã© de suma importancia
para que todos os aparelhos se adaptam corretamente com o tamanho imagem presente no site.

Criando uma imagem dinamica:

Para criar uma imagem em HTML5, basta usar a tag <img>, porem, essa imagem nao sera dinamica fazendo
com que algusn aparelhos tenham dificuldades em visualizar a imagem, por isso o conceito de imagens dinamicas.

Para criar uma imagem dinamica em HTML5 basta usar a tag <picture> e dentro da tag colocar entao a tag <img>.

Audios em HTML5
