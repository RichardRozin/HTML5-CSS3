Outras formatacoes

Codigo-fonte:

Para marcamos uma parte da frase em formato de codigo, basta usar a tag <code>.

EX: O comando document.getElementById('teste') é escrito na linguagem JavaScript.

Agora, para criarmos um codigo com mais de uma linha, precisamos colocar a tag <code> dento da tag <pre>.

    print(f'O numero {num é impar}.')
print('Fim do programa.')

Citacoes:

Para colocarmos uma frase como citacao (entre "") basta usar a tag <q>.

Como diria o pai de um amigo meu: O computador é um burro muito rapido.

Agora, se voce quiser fazer uma citacao mais completa, deslocando a citacao um pouco para a direita,

Abreviacoes:

Para dizer o significado de um determinado elemento em uma frase, basta usar a tag <abbr>.

EX: Estou aprendendo HTML e CSS. Estou gostando muito.
